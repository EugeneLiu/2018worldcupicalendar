# 2018worldcupicalendar

> 2018世界杯全部赛程的日历文件。

IOS设备用户可以通过使用Safari浏览器打开[链接](https://raw.githubusercontent.com/EugeneLiu/2018worldcupicalendar/master/2018worldcup.ics),将世界杯赛程添加到IOS日历软件中。

或者直接微信扫描二维码，然后使用Safari打开二维码指向的链接。
